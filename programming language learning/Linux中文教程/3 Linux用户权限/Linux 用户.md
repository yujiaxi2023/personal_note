**修改权限控制chmod**
![[Pasted image 20230419144417.png]]
chmod是可以修改文件的权限信息，但是只有在文件或者文件夹的所属用户或者root用户可以修改

语法就是chmod -R
-R可以让文件夹内的所有内容使用同样的操作

**示例**
chmod u=rwx,g=rx,o=x hello.txt
可以将文件修改为rwxr-x--x
