所以我们需要查看test 数据集的error
![[Pasted image 20230607163424.png]]

这个进行拟合的过程报告的test error很可能大于实际的误差,因为在训练集上出现的过拟合,在测试集上同样会出现过拟合测试集的问题
所以这个方式并不合适

可以将数据集变为,训练集,测试集,和交叉验证测试集
