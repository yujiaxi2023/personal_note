# vi/vim编辑器

快捷键分类
vim是vi的加强版而且快捷键是通用的

## 1. 移动光标：

h, j, k, l 左 下 上 有
ctrl-f 下翻一页 forward
ctrl-b 上翻一页 backward
ctrl-u 上翻半页 up
ctrl-d 下翻半页 down
0 跳到行首，不管有没有缩进，就是跳到0个字符
^ 跳到行首的第一个字符
$ 跳到行尾
gg 跳到文首
G 跳到文尾
5gg/5G 调到第5行

## 2. 删除复制：

- dd删除光标所在行，# 使用u撤销，可以还原到最初状态，使用ctrl + r恢复，dd其实是剪切
- 10dd是从光标往下删除10行
- dw删除一个单词（word）
- 小p是粘贴剪切板的内容到当前行的下面
- 大P是粘贴剪切板的内容到当前行的上边
- yy复制行
- 5yy复制5行
