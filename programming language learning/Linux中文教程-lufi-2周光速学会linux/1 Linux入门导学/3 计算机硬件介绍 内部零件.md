计算机：能够接受用户输入的指令和资料，并且通过计算机的CPU进行数学逻辑运算后产生有用的信息，通过输出设备进行有用信息展示

计算机构成部分：
- CPU
- 主记忆体（RAM）接收信息提供给cpu处理
- 硬盘，用于储存信息，也是CPU读写的地方
- 显卡，将各种图像输出的装置，由CPU将影像传到出来
- 主机板，连接各种原件
- 键盘，鼠标，网卡
