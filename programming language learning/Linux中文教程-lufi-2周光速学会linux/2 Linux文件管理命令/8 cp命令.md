### cp命令
```
cp copy

语法
cp [选项]... [-T] 源文件 目标文件
cp [选项]... 源文件... 目录
cp [选项]... -t 目录 源文件...
将源文件复制到目标文件,或者将多个源文件复制到目标目录

选项
-r 递归复制目录,复制目录下面所有层级的子目录和文件 -p 复制的时候保持属性不变
-d 复制的时候保持软连接(快捷方式)
-a 等于-pdr
-p 等于--preserve=模式,所有权,时间戳,复制文件的时候保持源文件的权限,时间属性
-i --interactive 覆盖前询问提示
```


```shell
# 1. 复制普通文件
cp命令 想要复制哪个文件 复制之后的文件名

# 2. 复制普通文件且改名,放入到另一个文件夹中
cp dms.txt ./oldboy # 复制放入其他文件夹不改名
cp dms.txt ./oldboy/mjj.txt2 # 复制到其他文件夹并且改名

# 3. 一次性复制多个文件,放入另一个文件夹
cp dms.txt dms2.txt ./dms

# 4. 复制整个文件夹,必须加上-r参数
cp -r greptut dms

# 5. 复制并保持文件属性不变,-p参数用法
cp -p dms.txt dms.txt1

# 6. 保持软连接不变,-d参数用法
cp -d link link1

# 7. -i参数用法,覆盖文件前提示,但是实际上不写也会提示,因为写了alias别名
# 实际上执行的cp -i
cp -i 文件1 文件2 # 如果文件2存在,就会覆盖
```
