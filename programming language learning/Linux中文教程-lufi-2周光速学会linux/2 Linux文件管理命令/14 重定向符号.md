| 符号               | 解释      |
| ---------------- | ------- |
| 重定向的意思是将数据传到其他地方 |         |
| >                | 输出覆盖重定向 |
| >>               | 输出追加重定向 |
| <或者<<            | 标准输入重定向 |

使用cat命令读取txt文件内容
这时候我们可以使用重定向将文件从a文件转移到b文件中

```shell
1. 读取文件内容,写入到另一个文件中,覆盖写入内容
cat yu.txt > yu1.txt

2. 追加写入内容
cat yu.txt >> yu1.txt

3. 重定向写入符号
cat  < yu.txt # 表示将文件中的数据,发送给cat命令数据

4. 将文本内容拆分成多行
xargs命令
xargs -n 3 < yu.txt # 这个是按照空格来确定的字符

5. 重定向追加写入符号
cat >> gushi.txt << EOF
> 床前明月光
> abcde
> 作者 yu
> EOF (代表END OF FILE)
```
