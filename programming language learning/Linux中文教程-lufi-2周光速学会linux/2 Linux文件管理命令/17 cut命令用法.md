
找到第n个字符对文件进行切割

```shell
cat yu2.txt
1. 截取第几个字符
cut -c 4 yu2.txt # 从第4个字符进行切割

2. 截取4-7个字符
cut -c 4-7 yu2.txt # 切割4-7个字符

3. 截取5和7个字符 
cut -c 4,6 yu2.txt # 切割4和6两个字符

4. 截取7个字符以前的内容范围
cut -c -7 yu2.txt # 切割7字符以前的所有字符

5. 截取8个字符以后的内容范围
cut -c 8- yu2.txt # 切割8字符以后的所有字符
```


cut - 在文件的每一行中提取片段
在每个文件的行中,把提取的片段显示在标准输出STDOUT
```
语法
cut [参数] [数值区间] 文件

-b 以字节为单位分割
-n 取消分割多字节字符,和-b一起使用
-c 以字符为单位
-d 自定义分隔符,默认以tab为分隔符
-f 和-d一起使用,指定显示什么区域
N 第N个字节,字符或者字段,从1计数开始
N- 从第N个字节,字符或者字段直到行尾
N-M 从第N到第M(并包括第M)个字节,字符或者字段
-M 从第1到第M(并包括第M)个字节,字符或者字段
```

案例
```shell
6. 指定分隔符号进行截取,必须要添加-f的区域范围
cut -d ":" -f 7 password # 显示第7个区域的信息
cut -d ":" -f 1-3 password # 显示1-3个的区域范围
cut -d ":" -f 1-3 password | head -5 # 显示1-3个的区域范围,并选择前5行
```
