### ls命令

| 命令           | 空格  | 参数（可写可不写）              | 空格  | 文件，文件夹（可写可不写）                 |
| ------------ | --- | ---------------------- | --- | ----------------------------- |
| ls：列出文件夹中的内容 |     |                        |     | /opt 根目录下的opt文件夹              |
| ls           |     | -a all 显示所有的文件和隐藏文件    |     | /opt                          |
| ls           |     | -a                     |     | 文件什么都不写默认是输出当前文件夹的内容，默认是一个点 . |
| ls           |     |                        |     |                               |
| ls           |     | -a -l/-al 显示所有文件并且显示详情 |     |                               |

基本命令用法如上：
- 一般情况下，linux命令的参数，都是可选的，可写可不写，不同的参数作用不一样
- linux命令之间，必须至少有一个空格
