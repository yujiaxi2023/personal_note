date可以在命令行中查看系统时间
语法：date -d +格式化字符串
- -d按照给定的字符串显示日期，一般用于日期计算
- 格式化字符串：通过特定的字符串标记，控制显示的日期格式
	- %Y 年
	- %y 年份后两位数字 00~99
	- %m 月份 01~12
	- %d 日 01~31
	- %H 小时 00~23
	- %M 分钟 00~59
	- %S 秒 00~60
	- %s 自1970-01-01 00：00：00 UTC到现在的秒数
