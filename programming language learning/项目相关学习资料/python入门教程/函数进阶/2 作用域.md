作用域：变量的访问权限

```python
a = 10 # 全局变量 -> 全局作用域  
  
print(a)  
  
def func(): # 全局变量函数  
    b = 20 # 局部变量,局部作用域  
    print(a)  
  
func()  
# 此时如果调用b  
# print(b) # 这样会报错  
  
def func3():  
    func() # 局部的一个函数  
  
func3() # 此时可以调用外部的函数  
  
  
def func():  
    c = 10086  
    return c # 如果需要在函数外边访问到函数内部的东西,只能使用return  
  
c1 = func() # 这里只是返回了c的值,但是如果要显示就需要打印  
print(c1)
```

函数内部可以访问外部的空间，但是外部不能访问内部
