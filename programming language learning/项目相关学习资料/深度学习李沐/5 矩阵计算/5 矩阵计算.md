# 梯度

![[Pasted image 20230921144140.png]]

1. 如果xy均为标量，则求y的x导数是标量
2. 如果y是标量，x是向量，则求导结果是向量，反之y是向量，x是标量亦然
3. 如果xy均为向量，则求导结果是矩阵

![[Pasted image 20230921144549.png]]
因为行向量×列向量才等于标量，所以列向量的求导结果是转置矩阵的形状

![[Pasted image 20230921144727.png]]

**梯度是指向变化值最大的方向**
