**1. 减 计算差集**
```python
s1 = {"liu", "zhao", "pi"}
s1 = {"liu", "feng", "pi"}

s3 = s1 - s2
s4 = s2 - s1
print(s3)
print(s4)
```

**2. & 计算交集**
```python
s1 = {"liu", "zhao", "pi"}
s1 = {"liu", "feng", "pi"}

s3 = s1 & s2
print(s3)
```

**3. | 计算并集**
```python
s1 = {"liu", "zhao", "pi"}
s1 = {"liu", "feng", "pi"}

s3 = s1 | s2
print(s3)
```

**4. 长度**
```python
v = {"liu", "zhao", "nigu"}
data = len(v)
print(data)
```

**5. for循环**
```python
v = {"liu", "zhao", "nigu"}
for item in v:
	print(item)
```
因为集合中没有顺序,所以没法使用索引的方法提取出需要的数据
只能够使用for循环将所有的元素打印出
