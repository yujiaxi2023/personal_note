
列表属于容器，内部可以存放各种数据，所以支持列表的嵌套， 例如
```python
data = ["xie", "hai", ["hai", "zhao"], True, [11, 22, 33, 44], "song"]
```
对于嵌套的值，可以进行索引， 例如
```python
data = ["xie", "hai", ["hai", "zhao"], True, [11, 22, 33, 44], "song"]
print(data[0]) "xie"
print(data[2]) ["hai", "zhao"]
print(data[0][2]) "e"
print(data[2][-1]) "zhao"

data.append(666)
print(data) # ["xie", "hai", ["hai", "zhao"], True, [11, 22, 33, 44], "song", 666]

data[2].append("xie")
print(data) # ["xie", "hai", ["hai", "zhao", "xie"], True, [11, 22, 33, 44], "song", 666]

del data[-2]
print(data) # ["xie", "hai", ["hai", "zhao", "xie"], True, [11, 22, 33, 44], 666]

data[-2][1] = "alex"
print(data) # ["xie", "hai", ["hai", "zhao", "xie"], True, [11, "alex", 33, 44], 666]

data[2][0:2] = [999, 666]
print(data) # ["xie", "hai", [999, 666, "xie"], True, [11, 22, 33, 44], "song", 666]
```
