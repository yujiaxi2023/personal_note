| ---- | ---- |
| ctrl + k + f | 自动格式化代码 |
| ctrl + k + c | 自动注释代码 |
| ctrl + k + u | 取消注释代码 |
| F5 | 调试运行 |
| ctrl + shift + b | 编译不运行，所有项目 |
| ctrl + b | 编译不运行，单独项目 |
