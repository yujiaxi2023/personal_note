首先github上创建仓库my_repo
然后本地的文件夹my_repo
此时my_repo没有远程连接仓库
```bash
git remote add origin git@github.com:yujiaxi2023/my_repo.git
git remote -v # 查看是否连接仓库
```
指定分支名称为main
```bash
git branch -M main
git push -u origin main:main # 后边的main可以省略
```
main分支和远程origin的main分支连接起来
本地仓库的内容就推送到远程仓库中去了
比如github中进行修改,或者别的推送了
