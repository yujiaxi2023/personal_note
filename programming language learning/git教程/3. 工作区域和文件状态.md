Git本地数据管理有三个区域
*工作区 working directory .git 所在的目录*
*暂存区 Staging Area/Index .git/index*
*本地仓库 Local Repository .git/object*

工作区域就是操作的目录
暂存区域就是中间区域
本地仓库就是储存git代码和版本信息的主要区域，包含项目的版本历史和元数据

修改完工作区的文件之后
```git
git add
```
需要添加到暂存区，然后提交到本地仓库
```git
git commit
```

中间可以用git命令查看历史信息，回退等操作。
