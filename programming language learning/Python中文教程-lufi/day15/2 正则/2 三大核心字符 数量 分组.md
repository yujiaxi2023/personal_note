## 正则表达式

**1. 字符相关**
- `wupeiqi`匹配文本中的wupeiqi
```python
# wupeiqi匹配文本中的wupeiqi  
text = "你好wupeiqi,adlfjkajdfwupeiqi,按等级分类wupeiqiffff"  
data_list = re.findall("wupeiqi", text)  
print(data_list) # ['wupeiqi', 'wupeiqi', 'wupeiqi'] 可用于计算字符串中某个字符出现的次数
```

- `[abc]`匹配a或者b或者c字符
```python
text = "你2b好wupeiqi,akldjsflkadg,奥地利快放假了去我饿u好噶asdc贵族ha"  
  
data_list = re.findall("[abc]", text)  
print(data_list) # ['b', 'a', 'a', 'a', 'c', 'a']  
  
data_list = re.findall('q[abc]',text)  
print(data_list) # [] 只有带q后边是abc才会提取
```

- `[^abc]`匹配除了abc之外的字符
```python
# [^abc]匹配除了abc之外的字符  
text = "你alkdjfjzckvj"  
data_list = re.findall("[^abc]",text)  
print(data_list) # ['你', 'l', 'k', 'd', 'j', 'f', 'j', 'z', 'k', 'v', 'j']
```


- `[a-z]` 匹配a-z的任意字符,`[0-9]`也可以,反过来`[9-0]`不行,字母一样
```python
text = "adljghanvcznadjf"  
data_list = re.findall("[a-z]", text)  
print(data_list) # ['a', 'd', 'l', 'j', 'g', 'h', 'a', 'n', 'v', 'c', 'z', 'n', 'a', 'd', 'j', 'f']  
  
# text = "1234013"  
# data_list = re.findall("[9-0]",text) 反向排序是不行的
```


- `. `代指除了换行符之外任意字符
```python
text = "alkdjfadfaklghoiujzm,cvnhgqpoijfavnqngodm,arfijeqo"  
data_list = re.findall("r.",text)  
print(data_list) # ['rf']  
  
text = "rpprppooroorooroasopijfdroo"  
data_list = re.findall("r.+o",text) # +代表出现1次或者n次 贪婪匹配  
print(data_list) # ['rpprppooroorooroasopijfdroo']  
# 默认遵循贪婪匹配,能更长就会更长,也就是如果有rooroo,就会都显示,而不是只有一个roo  
  
text = "rpprppooroorooroasopijfdroo"  
data_list = re.findall("r.+?o",text) # 非贪婪匹配  
print(data_list) # ['rpprppo', 'roo', 'roo', 'roaso', 'roo']
```

- `\w` 代指字母或者数字或者下划线(汉字)
```python
# \w 代指字母或者数字或者下划线(汉字) + 代表出现1次或者n次,并且贪婪匹配  
text = "北京wupeiqi,吴佩琦alex,北京  beijing 无 alex沛奇"  
data_list = re.findall("吴\w+x", text) # 因为中间有空格 所以能到最后,只能是中间是字母数字下划线才能继续  
print(data_list) # ['吴佩琦alex']
```

- `\d`代指数字
```python
# \d 代指数字  
text = "root-ad32min-add3-admdlin"  
data_list = re.findall("d\d",text) # 匹配到d和1个数字  
print(data_list) # ['d3', 'd3']  
  
data_list = re.findall("d\d+", text) # d和后边的整数,并且是贪婪匹配  
print(data_list) # ['d32', 'd3']
```

- `\s`代表任意空白符,空格或者制表符
```python
# \s 代表任意空白符,包括空格,制表符等  
text = "root admin add admin"  
data_list = re.findall("a\w+\s\w+",text) # a 后边字母数字下划线贪婪 然后是空白符制表符后字母数字下划线贪婪  
print(data_list) #['admin add'] 注意这里是a后边所有的字母数字下划线碰到空格因为\s继续,然后字母数字下划线,第二次空格就没法继续
```


### 数量相关

- `*` 重复0次或者更多次
```python
import re  
  
# * 重复0次或者更多次  
text = "他是大B个,确实是个大2B。"  
data_list = re.findall("大2*B", text) # 可以出现大B,大2B,大222B,2可以有无数个  
print(data_list) # ['大B', '大2B']
```

- `+` 重复1次或者更多次
```python
# + 重复1次或更多次  
text = "他是大B个,确实是个大2B, 大3B, 大66666B。"  
data_list = re.findall("大\d+B",text) # 大 加 1个或者n个数字 B,所以大B不可以因为没有数字  
print(data_list) # ['大2B', '大3B', '大66666B']
```

- `?` 重复0次或者1次
```python
# ? 重复0次或者1次  
text = "他是大B个,确实是个大2B, 大3B, 大66666B。"  
data_list = re.findall("大\d?B",text) # 大 加 0次或者1次数字 B,所以可以有大B 或者 大 1个数字 Bprint(data_list) # ['大B', '大2B', '大3B']
```

- `{n}` 重复n次
```python
# {n} 重复n次  
text = "楼主太牛逼了,在线想要192030291834@qq.com和xxxxx@live.com谢谢楼主,手机号也可12390095713034,搞起来啊"  
data_list = re.findall("123\d{5}",text) # 前缀 后边出现5位数的数字  
print(data_list) # ['12390095']
```

- `{n,}` 重复n次或者更多次
```python
# {n,} 重复n次或者更多次  
text = "楼主太牛逼了,在线想要192030291834@qq.com和xxxxx@live.com谢谢楼主,手机号也可12390095713034,搞起来啊"  
data_list = re.findall("\d{9,}",text) # 前缀数字 加 出现大于等于9次  
print(data_list) # ['192030291834', '12390095713034']
```

- `{n,m}` 重复n到m次
```python
# {n,m} 重复n到m次  
text = "楼主太牛逼了,在线想要192030291834@qq.com和xxxxx@live.com谢谢楼主,手机号也可12390095713034,搞起来啊"  
data_list = re.findall("\d{10,15}",text) # 10-15位的数字  
print(data_list) # ['192030291834', '12390095713034']
```


### 括号(分组)

- 提取数据区域,提取匹配成功的区域的指定文本
```python
# 提取数据区域,提取匹配成功区域的指定文本  
text = "楼主太牛逼了,在线想要192030291834@qq.com和xxxxx@live.com谢谢楼主,手机号也可12390095713034,1239132408510934搞起来啊"  
data_list = re.findall("123(9\d{5})",text) # 先从1239后边, 注意一定1239要连着的,因为如果是1230,就会直接找1230  
# 9 加 数字 5位,我只需要9 加 数字 这五位,不需要一长串  
print(data_list) # ['900957', '913240'] 6位数字  
  
data_list = re.findall("1(2)3(9\d{5})",text) # 匹配还是一样的的结果,提取数据还是一样  
print(data_list) # [('2', '900957'), ('2', '913240')] 获得一个列表中套元组,有几个搜索结果就几个元组,元组中按照几个括号来做几个元素  
  
text = "楼主太牛逼了,在线想要192030291834@qq.com和xxxxx@live.com谢谢楼主,手机号也可12390095713034,搞起来啊"  
data_list = re.findall("(123(9\d{5}))",text) # 外部括号套小括号,提取数据是按照括号提取,有几个括号元组就有几个元素  
# 所以这里应该是两个数据  
print(data_list) # [('123900957', '900957')]
```

- 获取指定区域+|条件(或者)
```python
# 获取指定区域+条件  
text = "楼主123r1234太牛逼了,在线想要123ryujiaxi192030291834太@qq.com太和xxxxx@live.com谢谢楼主,手机号也可12390095713034,搞起来啊"  
data_list = re.findall("123(9\d{5}|r\w+太)",text) # |是或者的意思,所以是9后边5个数可以成功,r后边是字母数字下划线+太  
print(data_list) # ['r1234太', 'ryujiaxi192030291834太', '900957']  
  
text = "楼主123r1234太牛逼了,在线想要123ryujiaxi192030291834太@qq.com太和xxxxx@live.com谢谢楼主,手机号也可12390095713034,搞起来啊"  
data_list = re.findall("(123(9\d{5}|r\w+太))", text)  
print(data_list) # [('123r1234太', 'r1234太'), ('123ryujiaxi192030291834太', 'ryujiaxi192030291834太'), ('123900957', '900957')]
```
