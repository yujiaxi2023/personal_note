```python
# 2. 面试题  
def num():  
    return [lambda x: i * x for i in range(4)]  
  
result = [m(2) for m in num()]  
print(result)  
"""  
1. 先执行num（）获得返回值  
2. for循环返回值  
3. 循环的每个元素，每个元素后加（2）  
"""  
"""  
1. 执行num（），会for循环4次，得到函数的列表[lambda x: i*x，···]，这时候for循环结束了，所以i = 3  
2. 所以m代指这个函数内存地址，输入形参2，执行函数，所以就是2*3=6  
3. 因为m是有4个函数被拿出来了，执行4次，就是[6,6,6,6]  
"""  
  
def num():  
    return (lambda x: i * x for i in range(4))  
  
result = [m(2) for m in num()]  
print(result)  
"""  
1. 先执行num（）获得返回值  
2. for循环返回值  
3. 循环的每个元素，每个元素后加（2）  
"""  
"""  
这里num执行的结果是生成器对象  
循环生成器的时候  
第一次是i = 0，返回值是函数，此时执行这个函数，传入的参数是2，所以结果是0  
第二次是i = 1，返回值是函数，执行函数，传参2，结果是2  
同理可得4，6作为第三第四次循环  
"""
```
用括号包围的一个推导式就是生成器表达式
其特点就是逐个生成元素，也就是使用for next或者.send来执行
