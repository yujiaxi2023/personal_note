给一个函数，在不修改函数源码的情况下，实现函数执行前后分别输入before和after

```python
def func():  
    print("我是func函数")  
    value = (11,22,33,44)  
    return value  
  
result = func()  
print(result)
```

## 简单入门

你的实现思路

```python
def func():  
    print("before")  
    print("我是func函数")  
    value = (11,22,33,44)  
    print("after")  
    return value  
  
result = func()  
print(result)
"""before  
我是func函数  
after  
(11, 22, 33, 44)"""
```

下面的做法是让func改变成为了inner

```python
def func():  
    print("before")  
    print("我是func函数")  
    value = (11, 22, 33, 44)  
    print("after")  
    return value  
  
  
result = func()  
print(result)  
  
  
def outer():  
    def inner():  
        pass  
  
    return inner  
  
  
func = outer()  # func = inner  
result = func()  # operating inner  
print(result)
```

```python
def func():  
    print("我是func函数")  
    value = (11, 22, 33, 44)  
    return value  
  
  
  
def outer(origin):  
    def inner():  
        res = origin() # 这样就会执行之前存储的一个func函数  
        return res  
  
    return inner  
  
  
func = outer(func)  # func就是原来的函数传进去了，就在outer中维护了一个原来的func函数  
result = func()  # operating inner  
print(result)
```
这个时候执行结果就跟原来是一模一样的
所以我可以在inner里面添加before和after
```python
def func():  
    print("我是func函数")  
    value = (11, 22, 33, 44)  
    return value  
  
  
  
def outer(origin):  
    def inner():  
        print("before")  
        res = origin() # 这样就会执行之前存储的一个func函数  
        print("after")  
        return res  
  
    return inner  
  
  
func = outer(func)  # func就是原来的函数传进去了，就在outer中维护了一个原来的func函数  
result = func()  # operating inner  
print(result)
```
