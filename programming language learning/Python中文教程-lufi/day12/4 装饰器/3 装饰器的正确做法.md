装饰器写法
```python
def outer(origin):  
    def inner():  
        print("before")  
        res = origin() # 调用原来的func函数并存储到outer空间中  
        print("after")  
        return res  
    return inner  
  
@outer  
def func_1():  
    print("我是func1函数")  
    value = (11,22,33,44)  
    return value  
  
@outer  
def func_2():  
    print("我是func2函数")  
    value = (11,22,33,44)  
    return value  
  
@outer  
def func_3():  
    print("我是func3函数")  
    value = (11,22,33,44)  
    return value  
  
func_1()  
func_2()  
func_3()
```
这种方式就比较简洁
而且维护成本高，需要修改每一处修改的地方，特别是重复的情况下
