## 后代选择器
```html
<!DOCTYPE html>  
<html lang="en">  
<head>  
    <meta charset="UTF-8">  
    <title>Title</title>  
  
    <style type="text/css">  
        #layer1 {  
            height: 500px;  
            width: 500px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed blue;  
        }  
  
        #layer2 {  
            height: 300px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed red;  
        }  
  
        #layer3 {  
            height: 200px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed black;  
        }  
  
        #layer1 p {  
            background-color: yellow; /*这个就是后代选择器*/  
        }  
    </style>  
</head>  
<body>  
  
<div id="layer1">  
    <p>layer1</p>  
    <h1>layer1 h1</h1>  
    <div id="layer2">  
        <p>layer 2</p>  
        <h2>layer 2 h2</h2>  
        <div id="layer3">  
            <p>layer3</p>  
            <h3>layer 3 h3</h3>  
        </div>    </div></div>  
<p>not in any layer</p>  
  
</body>  
</html>
```

## 子元素选择器
```html
<!DOCTYPE html>  
<html lang="en">  
<head>  
    <meta charset="UTF-8">  
    <title>Title</title>  
  
    <style type="text/css">  
        #layer1 {  
            height: 500px;  
            width: 500px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed blue;  
        }  
  
        #layer2 {  
            height: 300px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed red;  
        }  
  
        #layer3 {  
            height: 200px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed black;  
        }  
  
        #layer1 > p {  
            background-color: yellow; /*这个就是后代选择器*/  
        }  
    </style>  
</head>  
<body>  
  
<div id="layer1">  
    <p>layer1</p>  
    <h1>layer1 h1</h1>  
    <div id="layer2">  
        <p>layer 2</p>  
        <h2>layer 2 h2</h2>  
        <div id="layer3">  
            <p>layer3</p>  
            <h3>layer 3 h3</h3>  
        </div>    </div></div>  
<p>not in any layer</p>  
  
</body>  
</html>
```
这里是只对layer1下面的p进行操作，想要通过1对2进行操作是可以的，但是一般来说比较麻烦，可以直接对layer2进行操作

## 相邻兄弟选择器

可以选择紧接在另一个元素后边的元素，并且二者有相同的父元素

```html
<!DOCTYPE html>  
<html lang="en">  
<head>  
    <meta charset="UTF-8">  
    <title>Title</title>  
  
    <style type="text/css">  
        #layer1 {  
            height: 500px;  
            width: 500px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed blue;  
        }  
  
        #layer2 {  
            height: 300px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed red;  
        }  
  
        #layer3 {  
            height: 200px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed black;  
        }  
  
        #layer1 > p {  
            background-color: yellow; /*这个就是后代选择器*/  
        }  
  
        /*如果想要只给第二层的p进行样式，方便的做法是直接用第二层来找，用layer1找比较不方便*/  
        #layer2 > p + h2 {  
            background-color: saddlebrown;  
        }  
    </style>  
</head>  
<body>  
  
<div id="layer1">  
    <p>layer1</p>  
    <h1>layer1 h1</h1>  
    <div id="layer2">  
        <p>layer 2</p>  
        <h2>layer 2 h2</h2>  
        <div id="layer3">  
            <p>layer3</p>  
            <h3>layer 3 h3</h3>  
        </div>    </div></div>  
<p>not in any layer</p>  
  
</body>  
</html>
```
注意：必须是相邻的后边的一个元素选择，前边的和后边一个之后的元素都不会选择到


## 组合选择器

```html
<!DOCTYPE html>  
<html lang="en">  
<head>  
    <meta charset="UTF-8">  
    <title>Title</title>  
  
    <style type="text/css">  
        #layer1 {  
            height: 500px;  
            width: 500px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed blue;  
        }  
  
        #layer2 {  
            height: 300px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed red;  
        }  
  
        #layer3 {  
            height: 200px;  
            padding: 30px; /*内边距*/  
            border: 1px dashed black;  
        }  
  
        #layer1 > p {  
            background-color: yellow; /*这个就是后代选择器*/  
        }  
  
        /*如果想要只给第二层的p进行样式，方便的做法是直接用第二层来找，用layer1找比较不方便*/  
        #layer2 > p + h2 {  
            background-color: saddlebrown;  
        }  

		h2, p, h3 {
			background-color: black;
		}
    </style>  
</head>  
<body>  
  
<div id="layer1">  
    <p>layer1</p>  
    <h1>layer1 h1</h1>  
    <div id="layer2">  
        <p>layer 2</p>  
        <h2>layer 2 h2</h2>  
        <div id="layer3">  
            <p>layer3</p>  
            <h3>layer 3 h3</h3>  
        </div>    </div></div>  
<p>not in any layer</p>  
  
</body>  
</html>
```
