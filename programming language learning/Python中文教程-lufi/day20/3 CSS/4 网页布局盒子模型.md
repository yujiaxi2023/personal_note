使用div包起来的东西

- 内边距 padding
- 外边距 margin
![[Pasted image 20231203202826.png]]
- border 边框
- content 内容

```html
<!DOCTYPE html>  
<html lang="en">  
<head>  
    <meta charset="UTF-8">  
    <title>Title</title>  
  
    <style type="text/css">  
        #box_1 {  
            border: 10px solid black;  
            width: 500px;  
            height: 500px;  
            padding: 50px; /*这个时候会发现box整体大了*/  
            /*这时候我们可以打开审查元素。可以看到可视化的盒子模型，设置的时候生成的是内容的长宽，设置padding是在content外设置的*/  
            margin: 50px; /*四面都是多了50px*/  
        }  
        #box_2 {  
            border: 10px solid blue;  
            width: 500px;  
            height: 500px;  
        }  
    </style>  
</head>  
<body>  
  
<div id="box_1">  
  
    <p>hello</p>  
  
</div>  
  
<div id="box_2">  
  
</div>  
  
</body>  
</html>
```
padding
margin都是有上下左右四个分支命令的
注意盒子模型是一开始设置的width和height是content的大小
