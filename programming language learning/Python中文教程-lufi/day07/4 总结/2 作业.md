3. 循环提示用户输入，并将输入内容添加到字典中（如果输入N或n则停止循环）
```python
"""
例如：用户输入x1|wupeiqi, 则需要在字典中添加键值对{'x1':'wupeiqi'}
"""
```

```python
dic = {}  
  
while True:  
    content = input("please input your number and name (N/n stop) :")  
    if content.upper() == "N":  
        break  
    elif "|" not in content:  
        print("please input in this format(xx|xxxx)")  
    else:  
        data_list = content.split('|')  
        dic[data_list[0]] = data_list[1]  
    print(dic)
```

8. 有字符串"k: 1 |k1:2|k2:3 |k3:4"处理成字典{'k1':1,'k':2...}

```python
result = {}  
data = "k: 1 |k1:2|k2:3 |k3:4"  
data = data.split('|')  
  
for item in data:  
   data_list = item.split(':')  
   result[ data_list[0] ] = int(data_list[1].strip())  
  
print(result)
```

```python
v = "k1:3"
```
