基于python内置的shutil模块可以实现对压缩文件的操作
```python
import shutil  
  
# 1. 压缩文件  
# base_name, 压缩后的压缩包文件  
# format，压缩的格式，例如：”zip“，”tar”，“gztar”，“bztar”，or“xztar”  
# root_dir, 要压缩的文件夹路径  
shutil.make_archive(base_name=r"C:\Users\student\PycharmProjects\day09\venv", format="zip", root_dir=r"C:\Users\student\PycharmProjects\day09\venv")  
  
# 2. 解压文件  
# filename，要解压的压缩包文件  
# extract_dir 解压的路径  
# format，压缩文件格式  
shutil.unpack_archive(filename=r"C:\Users\student\PycharmProjects\day09\venv.zip", extract_dir=r"C:\Users\student\PycharmProjects\day09\venv", format="zip")  
# 目录这个可以写多层目录，并且都可以进行创建操作`
```
