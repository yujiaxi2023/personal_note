**转换**
其他类型转换为元组 用tuple套出
```python
name = "wupeiqi"
data = tuple(name)
print(data) # ("w", "u", "p", "e", "i",...)
```
```python
name = ["wupeiqi", 18, "pythonav"]
data = tuple(name)
print(data) # ("wupeiqi", 18, "pythonav")
```
只有str和list 现在是可以转换为元组的

**嵌套**
由于元组和列表都可以充当容器，且可以放任意元素，所以支持各种嵌套
```python
tu = ('jintianjeijiebuzaijia', 'jeifuxiaoyiziliaotian', ('jiefuwenxiaoyizishuihouduoshaoqian', 'xiaoyizishuohaitiqian'))
tu1 = tu[0]
tu2 = tu[2]
tu3 = tu[2][0]
tu4 = tu[2][1]
tu5 = tu[2][1][3]
```
