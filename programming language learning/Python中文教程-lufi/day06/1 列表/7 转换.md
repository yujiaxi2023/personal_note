- int bool 无法转换为列表
- str 转换没有意义
```python
name = "wupeiqi"
data = list(name) # ["w", "u", "p", "e", "i", "q", "i"]
```
开发中很少用到
- 元组转换为列表
```python
v1 = (11, 22, 33, 44) # 元组类型

list(v1) # [11, 22, 33, 44]
```
元组中的元素不可以修改，变为列表可以修改
- 集合转换为列表
```python
v2 = {"alex", "eric"} # 集合
list(v2) # ["alex", "eric"]
```
