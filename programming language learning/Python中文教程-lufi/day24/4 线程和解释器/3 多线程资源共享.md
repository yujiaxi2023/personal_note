
多进程和多线程的速度比较
```python
import time  
from multiprocessing import Process  
from threading import Thread  
  
def func(a,b):  
    return a,b  
  
if __name__ == '__main__':  
    t1 = time.time()  
    for i in range(500):  
        Thread(target=func, args=(10,20,)).start()  
  
    t2 = time.time()  
  
    t3 = time.time()  
    for i in range(500):  
        Process(target=func, args=(10,20, )).start()  
  
    t4 = time.time()  
    print(t2-t1) # 0.04414510726928711 执行500个线程  
    print(t4-t3) # 19.375316381454468 执行500个进程
```
进程开启的数量，不能够无上限的开启，一般维持在CPU核数的1-2倍

多线程共享进程资源
同一个进程下的所有线程是共享进程资源的，代码
```python
import time, random  
from threading import Thread  
  
# 全局变量  
num = 100  
  
def func():  
    global num  
    num -= 1  
  
if __name__ == '__main__':  
    thread_list = []  
    for i in range(10):  
        t = Thread(target=func) # 如果是进程，因为进程之间是隔离的，所以num是全局的  
        t.start()  
        thread_list.append(t)  
    t2 = time.time()  
    for t in thread_list:  
        t.join()  
  
    print(num) # 线程就可以拿到全局变量数据，就可以进行改变数据
```
