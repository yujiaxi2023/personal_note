如果要安装的模块在pypi.org不存在,或者因为特殊原因无法通过pip install安装的时候,可以直接下载源码,基于源码进行安装,例如:
- 下载requests源码(压缩包zip tar tar.gz)并解压
```
下载地址: https://pypi.org/project/requests/#files
```
中间一定有setup.py文件

- 进入目录
- 执行编译和安装命令
```
python3 setup.py build
python3 setup.py install
```
