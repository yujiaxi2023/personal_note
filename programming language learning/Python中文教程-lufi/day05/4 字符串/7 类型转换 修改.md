**3.4 转换**
```python
num = 999
data = str(num)
print(data) # "999"
```
布尔转换字符串没什么意义 
列表字典转换为字符串意义也不大
```python
data_list = ["alex","eric",999]
data = str(data_list)
print(data) # '["alex","eric",999]'
```
一般情况下这一串没有什么用
