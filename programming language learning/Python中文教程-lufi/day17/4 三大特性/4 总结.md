- 封装,将方法封装到类中 或 将数据封装到对象中,方便之后使用
- 继承,将类中的公共的方法提取到基类中实现
- 多态,python默认支持多态(这种方式成为鸭子类型),最简单的下面的代码就可以解释
```python
def func(arg):  
    v1 = arg.copy()  
    print(v1)  
  
  
func("wupeiqi")  
func([11,22,33,44,11])  
  
class Email(object):  
    def send(self):  
        print("send email")  
  
class Message(object):  
    def send(self):  
        print("send message")  
  
def func(arg):  
    v1 = arg.send()  
    print(v1)  
  
v1 = Email()  
func(v1)  
  
v2 = Message()  
func(v2)
```
